Simple Web Page with HTML and JavaScript

By: Andy for INET 4031

JavaScript can change HTML attribute values.

This website uses JavaSript and HTML. HTML is used to display the text on the website. Think of it like typing into a Word document and using Bold or Underline. It tells the browser how to format the text.

JavaScript allows the user to interact with the buttons below. It tells the browser what to do when the user selects between the two buttons.

In this case JavaScript changes the value of the src (source) attribute of an image.
